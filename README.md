# Advanced Machine Learning with Neural Networks

Homework assignments for the advanced ML course at Chalmers University, covering deep learning methods implemented from scratch in Python.

## Contents

### Homework 1 – Neural Networks from Scratch
- Single artificial neuron with sigmoid activation (1D and 2D)
- Gradient descent and backpropagation implemented manually
- Batch-wise training loop
- Two-layer and three-layer neural networks
- Function approximation with tanh activation

## Stack
- Python, NumPy, Matplotlib
